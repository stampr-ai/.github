## 🚀 Overview

`stampr_ai` is a lightweight system for fingerprinting AI models in a blackbox setting. It enables:
- **Behavioural fingerprint extraction**
- **Version tracking over time**
- **Public verification of model identity**

We provide:
- A **web-based tracker** for visualising fingerprint evolution
- A **Python package** for local verification
